# How to Submit

This assignment is distributed via **GitHub Classroom**. When you accept the assignment invite, GitHub creates a *private copy of this repo* under your own account — only you and the instructor can see it.

1. Accept the assignment link shared by the instructor. This creates your personal repo, e.g. `KRMU_LLM_DL_AS1-<your-github-username>`.
2. Clone your repo (not this one):
   ```bash
   git clone https://github.com/<org-or-username>/KRMU_LLM_DL_AS1-<your-username>.git
   ```
3. Do your work in `assignment/starter_notebook.ipynb`. Keep the section headers intact.
4. Commit and push to your repo before the deadline:
   ```bash
   git add assignment/starter_notebook.ipynb
   git commit -m "Complete assessment 1"
   git push
   ```
5. That's it — the last commit on your default branch before the deadline is your submission. No pull request needed.

Do not fork or push to this template repo directly — it is read-only for students.
